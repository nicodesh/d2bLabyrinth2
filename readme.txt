-------------
Contraintes :
-------------

- Le programme est uniquement compatible avec Python 3.
- Je l'ai développé sous Mac OS X, aussi je ne peux garantir son fonctionnement que sur Mac OS X et Linux.

--------------
Spécificités :
--------------

- Tous les commentaires sont en anglais afin de faciliter la maintenance du code pour plus tard.
- Le jeu en soi présente peu d'utilité, mais il me semble être une bonne base pour plein de jolies choses, d'où mon choix :)

---------------------------
Pour lancer l'application :
---------------------------

- Côté serveur : exécuter python3 server.py
- Côté client : exécuter python3 client.py

---------------------------------
Pour lancer les tests unitaires :
---------------------------------

1. Se placer à la racine du projet
2. Exécuter la commande suivante : python3 -m unittest
